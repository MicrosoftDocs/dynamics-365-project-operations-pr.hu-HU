<span data-ttu-id="1a9d8-104">A projektmenedzserek munkalebontási struktúra (WBS)-sablonokat állíthatnak be, amelyek akkor alkalmazhatók, amikor új projektekhez hoznak létre WBS-t.</span><span class="sxs-lookup"><span data-stu-id="1a9d8-104">Project managers can set up Work breakdown structure (WBS) templates that they can apply when they create a WBS for new projects.</span></span> <span data-ttu-id="1a9d8-105">A projektmenedzserek szerepköröket vehetnek fel, amikor létrehoznak egy sablont.</span><span class="sxs-lookup"><span data-stu-id="1a9d8-105">Project managers can add roles when they create a template.</span></span> <span data-ttu-id="1a9d8-106">A következő eljárással lehet szerepkört rendelni egy WBS-sablonhoz.</span><span class="sxs-lookup"><span data-stu-id="1a9d8-106">Use the following procedure to assign a role to a WBS template.</span></span>
